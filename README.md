# Food-Insecurity-in-DC

## Overview 
This story is a comparative analysis of data on DC grocery stores from 2015-2025. 
## Data Sources 
- This data is from the DC open data database from the Office of the Chief Technology Officer.
## Methodology 
- Added an identified issue column for each year
- Fixed the headings in place
- Added a filter to sort by Ward
- Looked for patterns and conducted research (with included links) to fill in data gaps (ex: if a store was open in 2018 and in 2022 it can be assumed it was open during 2019-2021). 
- Excluded the years 1990-2014 to narrow the scope
- Excluded 8 grocery stores that were unable to have complete data
- To calculate stores per square mile, I used data from Censusreporter.org that uses data from the 2024 census to find the square miles in each ward. Then, I divided the number of open stores in 2015 and 2025 by the number of square miles.  
## Limitations 
- Small sample size
- Does not include other ways to get groceries
- Incomplete original data 
## How to Use 
- Use this data for data analysis, trends over time, and data visualization.
- Blue Highlight: I conducted the research.
- Gray Highlight: Years not included in data analysis.
- Yellow Highlight: Stores excluded from analysis (due to unverified opening and closing dates).
